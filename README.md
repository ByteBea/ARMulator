# ARMulator

## Project Description

This repository contains Python source files developed for educational purposes
as part of a collaborative university project.

The objective of the project is to study and implement selected concepts of the
ARMv7 architecture in a high-level programming environment, using Unicorn Engine (based on QEMU) with a focus on
understanding architectural behaviors.

The project is based on [ARMulator](https://github.com/Filippo2903/ARMulator), an ARMv4 assembler and emulator.

The implementation aim to be a didactic tool intended for learning, experimentation and analysis.
